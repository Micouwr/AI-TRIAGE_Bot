# Tools Directory

This directory contains various utility scripts designed to support the operation and maintenance of the AI Triage Bot Prototype.

## Contents

- **view_fallbacks.py**: Provides a human-readable view of the fallback log, aiding in debugging and monitoring.

## Purpose

These tools are intended to assist with common tasks such as log analysis, system monitoring, and performance tracking, enhancing the overall governance and management of the AI Triage Bot.

## How to Use

Each script can be run independently from the command line. Refer to each script's specific documentation for detailed usage instructions.

## List of Tools

- **view_fallbacks.py**: To view the fallback log, execute the following command:
  ```bash
 python tools/view_fallbacks.py
